# **App Name**: CampusAssist AI

## Core Features:

- FAQ Answering: Respond to user queries based on a knowledge base of college FAQs.
- College Data Training: Train the chatbot on college-specific data, including academic calendars, department information, and campus resources. A tool can use its discretion about how to use different available data.
- Multilingual Support: Provide responses and understand user input in multiple languages.
- User Interface: Clean, intuitive chat interface with clear question and response display.
- NLP Processing: Natural language processing (NLP) integration to understand and process user input effectively.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB) for a welcoming and informative feel.
- Background color: Very light grey (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Soft green (#90EE90) for positive feedback and highlighting important information.
- Font pairing: 'Inter' for body text (sans-serif, modern, neutral look) and 'PT Sans' for headlines (sans-serif, modern with personality).
- Use simple, recognizable icons for common helpdesk topics.
- Chat bubbles to differentiate user and bot messages; easy to read layout; and an option to make use of expandable sections in chatbot answers.
- Subtle animations for message loading and response display.
